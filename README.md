## 🔐 Método: Documentação Pública, Dados Privados

> Todo case neste portfólio segue o mesmo princípio: **a entrega real do cliente fica sob NDA, mas o raciocínio analítico é aberto.**

O que você encontra em cada repositório:

| ✅ Público no GitHub | 🚫 Nunca exposto |
|---|---|
| README com contexto de negócio | Dados reais de clientes |
| Modelagem dimensional & ERDs | Nomes, CPFs, contratos |
| Queries SQL e medidas DAX comentadas | Bases originais |
| Wireframes e prints anonimizados | Valores financeiros reais |
| Estrutura de KPIs e regras de negócio | Identificadores internos |

Resultado: você consegue **avaliar a engenharia analítica** sem que nenhum dado sigiloso saia da operação do cliente.
