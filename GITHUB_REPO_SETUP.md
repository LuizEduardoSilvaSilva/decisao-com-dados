# Configuração do Repositório no GitHub

## Descrição curta (campo "About" no topo do repo)

Copie e cole esta descrição no campo **About** do seu repositório no GitHub:

```
Portfólio de Análise de Dados & BI — cases com Power BI, SQL, DAX e Excel. Documentação pública, dados privados.
```

## Topics / Tags

Adicione estas tags no campo **Topics** do repositório:

- `data-analytics`
- `business-intelligence`
- `portfolio`
- `powerbi`
- `sql`
- `dashboards`
- `dax`
- `data-visualization`

## Como aplicar

1. Acesse a página principal do repositório no GitHub
2. Clique no ícone de engrenagem (⚙️) ao lado do campo "About" (canto superior direito)
3. Cole a descrição no campo **Description**
4. Adicione as tags no campo **Topics**
5. Clique em **Save changes**
